# Technology Stack: Nova

## Core Logic & Backend
*   **Language:** Python 3.8+
*   **AI Inference:** Ollama (Local LLM Serving)
*   **Orchestration:** Advanced PVEV (Plan-Validate-Execute-Verify) Loop with Dynamic Model Routing
*   **Knowledge & Memory:** Tiered Memory System (RAM history, ChromaDB Vector Store, Episodic Procedural Store)

## Interfaces
*   **CLI/TUI:** Textual & Rich (Cyberpunk Neon Theme)
*   **Web UI:** React 19, TypeScript, Vite
*   **API:** FastAPI (Serving the Agent capabilities)

## Infrastructure & Tooling
*   **Containerization:** Docker & Docker Compose
*   **Edge AI:** Google AI Edge (MediaPipe) for local vision
*   **Packaging:** Setuptools (pyproject.toml, setup.py)
*   **Testing:** Pytest (tests/)
