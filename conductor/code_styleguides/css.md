# CSS Style Guide

## 1. Naming
*   **BEM:** Use Block Element Modifier (BEM) convention for class names (e.g., `block__element--modifier`).
*   **kebab-case:** Use kebab-case for class names.

## 2. Formatting
*   **Indentation:** 2 spaces.
*   **Ordering:** Position -> Box Model -> Typography -> Visual -> Misc.

## 3. Units
*   Use `rem` for layout and font sizes.
*   Use `em` for component-relative sizing.
*   Avoid `px` for scalable properties.
