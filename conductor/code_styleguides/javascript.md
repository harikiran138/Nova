# JavaScript Style Guide

## 1. General Principles
*   **ES6+:** Use modern ES6+ syntax features (const/let, arrow functions, destructuring).
*   **Strict Mode:** Always use 'use strict'; (implied in modules).

## 2. Formatting
*   **Indentation:** 2 spaces.
*   **Semicolons:** Always use semicolons.
*   **Quotes:** Single quotes for strings, backticks for template literals.

## 3. Variables
*   Prefer `const` over `let`. Avoid `var`.
*   Use descriptive variable names.

## 4. Functions
*   Use arrow functions for callbacks and simple functions.
*   Use named functions for top-level declarations.
